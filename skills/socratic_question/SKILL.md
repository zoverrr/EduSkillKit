## 任务

请基于学生的回答，生成一套苏格拉底式追问序列。你的目标不是驳倒学生，而是帮助他/她通过自我反思走向更深层的理解。

### 一、假设探查（3-5个问题）

识别学生回答中隐含的、未经检验的假设。针对每个假设设计一个问题，让学生意识到自己的预设前提。

提问方向：
- 这个观点成立的前提是什么？
- 如果这个前提不成立，结论会怎样变化？
- 你是否默认了某个未被证明的条件？

### 二、证据审视（2-3个问题）

引导学生检验自己观点的支撑证据是否充分、可靠。

提问方向：
- 你用什么证据来支持这个判断？
- 这个证据的来源可靠吗？是否存在反例？
- 如果换一个角度看待同样的证据，会不会得出不同结论？

### 三、逻辑推演（2-3个问题）

帮助学生检验推理链条是否严密。

提问方向：
- 从A到B的推理过程中，是否跳过了某些步骤？
- 这个推论在所有情况下都成立，还是只在特定条件下成立？
- 如果把这个逻辑推广到其他情境，会出现什么问题？

{% if depth == "中级" or depth == "高级" %}
### 四、视角转换（2-3个问题）

引导学生站在对立面或不同立场重新审视问题。

提问方向：
- 如果持相反观点的人来看这个问题，他/她会怎么反驳你？
- 这个问题在不同文化、不同历史时期会有不同答案吗？
- 你能否为你不认同的观点找到合理之处？
{% endif %}

{% if depth == "高级" %}
### 五、元认知追问（1-2个问题）

引导学生反思自己的思维过程本身。

提问方向：
- 你是如何得出这个结论的？你的思考路径是什么？
- 在回答这个问题时，你的情绪或直觉是否影响了你的判断？
- 如果重新思考这个问题，你会改变思路吗？为什么？
{% endif %}

### 六、逻辑缺口分析

简要分析学生回答中的主要逻辑缺口或认知偏差：
- 指出最需要修正或深化的1-2个关键点
- 用建设性的语言说明学生思考中的亮点与不足
- 避免居高临下的评判，保持引导者的谦逊姿态

### 七、后续学习建议

根据学生当前的认知水平，推荐2-3项后续学习活动：
- 能帮助学生弥补认知缺口的具体阅读或实践任务
- 可以进一步深化理解的思考练习
- 适合与同学讨论或辩论的延伸话题

## 输出原则

- **不给答案**：所有问题都应该是开放式的，不能通过"是/否"简单回答
- **循序渐进**：从表层问题逐步深入，符合认知发展规律
- **尊重学生**：提问语气应该是好奇的探索，而非质问或审讯
