## 板书设计规范

### 板面分区

将黑板划分为三个区域：
- **左侧（主板区）**：约占板面50%，书写核心教学内容
- **右侧（副板区）**：约占板面30%，书写辅助内容、例题、补充说明
- **顶部（标题区）**：书写课程名称和章节标题

{% if style == "提纲式" %}
#### 提纲式板书布局特点

- 采用分级标题结构，层次清晰
- 主标题、副标题、要点逐级缩进
- 使用序号（一、二、三 / 1、2、3 / (1)(2)(3)）标识层级
- 适合知识体系性较强的教学内容

{% elif style == "图解式" %}
#### 图解式板书布局特点

- 以核心概念为中心，向外延伸分支
- 使用箭头、连线表示逻辑关系
- 适合展示概念之间的关联和知识网络
- 注意留足空间，避免线条交叉混乱

{% elif style == "表格式" %}
#### 表格式板书布局特点

- 使用表格对比或归纳知识要素
- 表头清晰，行列对应关系明确
- 适合对比分析、特征归纳类内容
- 预先画好表格框架，逐步填写内容

{% elif style == "对比式" %}
#### 对比式板书布局特点

- 板面分为左右或上下两部分
- 对比项逐条对应，一目了然
- 用对比线、色框等突出差异
- 适合易混淆概念的辨析

{% endif %}

### 三、书写时间线

按教学进程安排板书的书写时机：

| 教学时间 | 教学环节 | 板书动作 | 板书内容 | 操作说明 |
|----------|----------|----------|----------|----------|
| 0-3分钟 | 导入 | 书写标题 | | 课前可提前写好 |
| 3-15分钟 | 新授一 | 书写主板区第一部分 | | 边讲边写，同步进行 |

## 设计原则

- **整体性**：板书呈现完整的知识结构，下课时板面即为本课知识地图
- **简洁性**：内容精炼，每块板书区域信息量适中，避免过于拥挤
- **逻辑性**：内容之间的层级关系和逻辑联系清晰可见
- **生成性**：预留空间记录课堂生成的内容和学生精彩回答
- **实用性**：书写操作简便，不影响教学节奏
- **美观性**：字迹工整、布局匀称、色彩协调
